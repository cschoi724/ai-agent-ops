# Task Templates

작성일: 2026-06-29  
상태: Draft v1

## 1. 목적

이 폴더는 `.ai_project/tasks/`에 생성할 Task 파일 템플릿을 제공한다.

Task 파일은 Agent 실행 지시의 source of truth다. `task_board.md`와 handoff 문서는 Task 파일을 보조한다.

## 2. 템플릿

| 파일 | 용도 |
|---|---|
| `task.md` | 일반 Task 파일 템플릿 |

## 3. 파일명 규칙

```text
T-YYYYMMDD-001_task-slug.md
```
