# Report Templates

작성일: 2026-06-29  
상태: Draft v1

## 1. 목적

이 폴더는 `.ai_project/reports/`에 생성할 Development Agent 보고서 템플릿을 제공한다.

실행 지시의 source of truth는 `.ai_project/tasks/`의 Task 파일이다. 보고서는 Task 실행 결과를 기록하는 산출물이다.

## 2. 템플릿

| 파일 | 용도 |
|---|---|
| `dev_report.md` | Development Agent가 구현 또는 분석 완료 후 작성 |

## 3. 파일명 규칙

```text
T-YYYYMMDD-001_dev-report.md
```

|---|---|
