# Bootstrap

`bootstrap/`은 새 프로젝트 또는 기존 프로젝트에 AI Agent Ops를 적용할 때의 선택 절차를 담는다.

| 문서 | 역할 |
|---|---|
| `cold_start_prompt.md` | `.ai/`가 아직 없는 프로젝트에서 최초 1회 사용하는 시작 프롬프트 |
| `install_runbook.md` | 실제 프로젝트에 `.ai/` 운영체계 템플릿을 설치하는 절차 |
| `migration_runbook.md` | 기존 프로젝트의 운영모델 점검과 안전 적용 절차 |
| `project_bootstrap_policy.md` | AI Ops Agent가 선택지를 제시하고 `.ai_project` 구성을 확정하는 절차 |
| `bootstrap_runbook.md` | AI Ops Agent가 먼저 따르는 짧은 bootstrap 핵심 실행 흐름 |
| `bootstrap_reference.md` | 상세 선택지, 질문 팩, mode별 예시 |
